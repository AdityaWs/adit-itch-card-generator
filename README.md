# Adit's Itch Card Generator

**(Unreliable)** this is a simple Web App that i created to download my itch.io game card instead of screenshooting it for my instagram post. Due to CORS policy, the app may be unreliable and slow when it comes to loading image, so be cautious. I thought of releasing it here and making the code open in case you wanna use it and you can modify it to use your own proxy since this code uses an open public proxy which is slow

## Usage

To use the app, simply open [https://adit.madbyad.my.id/adit-itch-card-generator](https://adit.madbyad.my.id/adit-itch-card-generator) and enter the url of your itch.io project. waiting is required here due to the CORS issue

## ⚠️ Unreliable

Due to CORS (Cross Origin Resource Sharing), getting the thumbnail of your itch.io project will be slow and patience is required, and if you're not patient i recommending just screenshot from your itch.io page yourself. It's a tool for my itch.io marketing needs and i share it incase others may want to use it
